  \section{\textbf{Problema 3}}
  \noindent
  \begin{questions}

    \begin{enumerate}

    \item \textbf{\emph{30pts.}} Considere el siguiente pseudocódigo, que define una lista  enlazada de nodos:

    \texttt{struct Node \{}\\
    $~~~~$\texttt{data;} // Dato almacenado en el nodo\\
    $~~~~$\texttt{next;} // Puntero al nodo siguiente (NULL para el último nodo)\\
    \texttt{\}}\\
    \texttt{Punteros de la lista \{}\\
    $~~~~$\texttt{Node FirstNode;} // La lista apunta al primer nodo; NULL si está vacía\\
    $~~~~$\texttt{Node LastNode;} // La lista apunta al último nodo; NULL si está vacía\\
    \texttt{\}}

    Para insertar un nodo \texttt{newNode} después de un nodo \texttt{node}, se define la función \texttt{insertAfter}:

    \texttt{function insertAfter(Node node, Node newNode) \{}\\
    $~~~~$\texttt{newNode.next := node.next;}\\
    $~~~~$\texttt{node.next := newNode;}\\
    \texttt{\}}

    Para insertar un nodo \texttt{newNode} al inicio de la lista \texttt{list}, se define la función \texttt{insertBeginning}:

    \texttt{function insertBeginning(List list, Node newNode) \{}\\
    $~~~~$\texttt{newNode.next   := list.firstNode;}\\
    $~~~~$\texttt{list.firstNode := newNode;}\\
    \texttt{\}}

    A partir de lo anterior, defina:
    \begin{enumerate}
        \item Una función \texttt{insertEnd}, que inserta un nodo \texttt{newNode} al final de \texttt{list}.  [15 pts]
        \item Una función \texttt{removeAfter}, que elimina un nodo \texttt{Node}. [15 pts]
    \end{enumerate}
\end{enumerate}

      \end{enumerate}
    \end{questions}
